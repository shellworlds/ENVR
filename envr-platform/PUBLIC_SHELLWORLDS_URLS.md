# Public URLs (github.com/shellworlds only)

**Invalid forever:** `https://github.com/shellworlds/ZENVR/ZENVR91` — GitHub has no such path. Public or private makes no difference.

**Use instead:** `https://github.com/shellworlds/ZENVR` (after setup, default branch is ZENVR91) or `https://github.com/shellworlds/ZENVR/tree/ZENVR91`.

See **ACCESS_GITHUB.md** and **docs/URL_FORMAT_AND_ACCESS.md**.

All repositories below must be **public** on GitHub. Run `scripts/setup_shellworlds_public_all.sh` after `gh auth login` to create missing repos and push.
