# Kryptur PT-OF 1.6 Programmable Tunneling Optimized Filter

Full-stack quantum noise suppression subsystem with AI-driven real-time flux locking.

## Stack
- Backend: Python (FastAPI, QCoDeS, PyTorch), Go (health), Java (FPGA interface), C++ (ring buffer)
- Frontend: React + Vite + Plotly.js
- Databases: MongoDB, PostgreSQL (Docker)
- Orchestration: asyncio + Preempt-RT

## Quick Start
1. ./install_deps.sh
2. docker-compose up -d
3. source ptof_env/bin/activate && uvicorn api.main:app --reload
4. cd web && npm install && npm run dev
