Support Module Proof

Theorem: For a ring A, an A-module M, and I = Ann(M),
the support Supp(M) is contained in V(I).

Interactive Demonstration







Proof Steps:

  1. Let p ∈ Supp(M) (so M_p ≠ 0)
  2. Assume for contradiction that I ⊈ p
  3. Then ∃a ∈ I such that a ∉ p
  4. Since a ∉ p, a is a unit in A_p
  5. But a ∈ I = Ann(M), so aM = 0
  6. Localizing: (a/1) acts as 0 on M_p
  7. Since a/1 is invertible, M_p = 0
  8. Contradiction! Therefore I ⊆ p
  9. Hence p ∈ V(I), so Supp(M) ⊆ V(I)
