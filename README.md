# ENVR Module Splitter Implementation

## Project Overview
Implementation of the Module Splitting Theorem: Proof that M = L ⊕ N with given homomorphisms α, β, σ, ρ iff βα=0, βσ=1, ρσ=0, ρα=1, αρ+σβ=1.
