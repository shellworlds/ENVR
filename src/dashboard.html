SLK8 Problem: Support of M = ℚ/ℤ

Multi-language implementation and analysis

Problem Statement

Let ℤ be the ring of integers, ℚ the rational numbers, and set M := ℚ/ℤ. Find the support Supp(M), and show that it's not Zariski closed.

Mathematical Result

Support: Supp(M) = { (p) | p is prime }

Zariski Closed: No

Reason: Infinite set of nonzero primes, not finite and not whole space
